# Proyecto 6: De patria a patria (Homeland)

### Descripción del proyecto
Una página web responsiva que muestra las ciudades natales de algunos empleados de TripleTen, desarrollada y completada como parte del **Sprint 6**. 

En esta etapa final, el objetivo principal evolucionó para dominar la construcción de layouts bidimensionales complejos utilizando **CSS Grid**, implementar estados interactivos fluidos y perfeccionar la metodología **BEM** para mantener un código limpio y escalable. Además, se refinó el uso de **Media Queries** y el modelo de caja (Box Model) para asegurar una adaptabilidad fluida en cualquier dispositivo (desde resoluciones de escritorio hasta dispositivos móviles de 320px).

### Tecnologías y metodologías utilizadas
* **HTML5:** Marcado semántico y arquitectura limpia sin exceso de clases.
* **CSS3 Avanzado:**
  * **CSS Grid:** Uso de `grid-template-areas`, fracciones (`fr`) y `min-content` para diseños responsivos y fluidos.
  * **Media Queries:** Reestructuración dinámica de layouts según el punto de ruptura (breakpoints).
  * **Efectos e Interactividad:** Implementación de estados `:hover`, transiciones suaves (`transition`), manipulación de imágenes (`filter: brightness`), sombras (`box-shadow`) y fondos con gradientes lineales.
* **Metodología BEM:** (Block, Element, Modifier) aplicada rigurosamente para una alta mantenibilidad del código.
