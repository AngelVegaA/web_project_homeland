# Proyecto 5: De patria a patria (Homeland)

### Descripción del proyecto
Una página web responsiva que muestra las ciudades natales de algunos empleados de TripleTen, desarrollada como parte del Sprint 5. El objetivo principal fue practicar el uso de Figma, metodologías BEM y Media Queries para asegurar la adaptabilidad en dispositivos de 1280px, 768px y 320px.

### Tecnologías utilizadas
* HTML5 (Semántico)
* CSS3 (Flexbox, Media Queries)
* Metodología BEM
* Tipografía Inter (Carga local)
